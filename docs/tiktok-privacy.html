Privacy Policy

Effective date: May 21, 2026

1. Overview

YTS Render is a private publishing workflow. This policy explains how the tool handles data when its TikTok integration is used.

2. Data processed

When TikTok publishing is enabled, the service may process the TikTok access token granted by the account owner, creator information returned by TikTok's creator info endpoint, generated video files, captions, hashtags, publication status, publishing identifiers, and error messages returned by TikTok.

3. Purpose of processing

The data is used only to publish approved videos to the authorized TikTok account, check publication status, avoid duplicate posting, and show operational status inside the private YTS Render hub.

4. Storage

Tokens and publishing metadata are stored in the operator's server environment and local application database or artifacts. The service does not sell, rent, or share TikTok data with advertisers or unrelated third parties.

5. Retention and deletion

Publishing metadata is retained as part of the operator's audit trail. TikTok access can be revoked by removing the access token from the server configuration or revoking the app's access from the TikTok account settings.

6. Public users

YTS Render does not offer public user registration and does not collect personal information from visitors to these static legal pages.

7. Contact

For privacy questions, contact the repository owner through the public project page: github.com/marcoscoelhov/yts-render.
