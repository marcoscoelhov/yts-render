TikTok authorization callback

This static page is used as the registered TikTok OAuth redirect URI for YTS Render.

After TikTok redirects back here, copy the code parameter from the browser address bar and exchange it for a user access token using TikTok's official OAuth token endpoint.

If the URL contains an error parameter, the authorization did not complete and the app configuration or requested scopes should be reviewed in TikTok for Developers.
